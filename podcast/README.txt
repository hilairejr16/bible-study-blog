UPLOAD YOUR ZOOM RECORDING AUDIO FILES INTO THIS FOLDER
========================================================

SUPPORTED FILE TYPES:
  .m4a  (Zoom audio-only recordings — recommended, smallest size)
  .mp3  (if you convert the recording)
  .mp4  (Zoom video recordings — larger file)

HOW TO NAME YOUR FILES:
  Use lowercase, hyphens, no spaces.
  Include the date so episodes stay organized.

  Examples:
    bible-study-2026-05-01.m4a
    brothers-session-2026-05-08.m4a
    armor-of-god-lesson-2026-05-15.mp3

FILE SIZE GUIDE:
  1-hour session saved as M4A  = roughly 20–50 MB  ✅ Fine for GitHub
  1-hour session saved as MP4  = roughly 200–600 MB ⚠ Too large for GitHub
  → Always prefer the audio-only (M4A) Zoom recording, NOT the video.

HOW TO FIND YOUR ZOOM AUDIO RECORDING:
  After a Zoom meeting ends, look in:
    Documents → Zoom → [Meeting Date Folder] → audio_only.m4a

  If you only see an .mp4 file, you can still upload it —
  but ask Claude to help you convert it to M4A first to save space.

AFTER UPLOADING:
  Open podcast.json on GitHub and add a new episode entry.
  See README.md in the main blog folder for the full template.
